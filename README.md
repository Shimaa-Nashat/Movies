# 🎬 Favorite Disney Movies

🚀 [Click Here to View the Project](https://shimaa-nashat.github.io/Movies/)

A **dynamic web page** showcasing favorite Disney movies. Each movie appears as a card with an image, title, short description, and a **Watch** button that opens the movie page in a new tab.

---

## 🛠 Features

- Dynamic movie cards generated using **JavaScript**
- Responsive layout using **Flexbox**
- Hover effects for movie cards
- Hover animation for buttons
- Dark-themed page design
- Header section with background image and title
- Easy to add new movies by editing arrays in `script.js`

---

## ⚙️ Technologies Used

- **HTML5**
- **JavaScript**
- **Flexbox (CSS layout)**
- **DOM Manipulation**

## 🚀 How to Run

1. Clone the repository:

```bash
git clone https://github.com/Shimaa-Nashat/Movies.git
```

2. Open the project folder.
3. Run the project by opening:

```
index.html
```

in your browser.

---

## 💡 How to Add More Movies

You can easily add new movies by updating the arrays in **script.js**.

Example:

```javascript
images.push("images/newMovie.png");
names.push("New Movie");
moviesDescriptions.push("Short description of the movie.");
links.push("https://example.com/movie");
```
